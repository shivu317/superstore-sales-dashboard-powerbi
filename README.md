## 🚀 Project Overview

An interactive **Power BI dashboard** built using the Superstore dataset to analyze sales performance, profitability trends, customer behavior, and regional performance.

The dashboard empowers decision-makers to identify growth opportunities, monitor KPIs, and make data-driven business decisions.

---

## 🎯 Objectives

* Analyze overall sales and profit performance
* Track regional and category-wise trends
* Identify top-performing products
* Monitor key business KPIs
* Enable interactive exploration through filters and slicers

---

## 🛠 Tech Stack

* 📊 Power BI Desktop
* ⚡ DAX
* 🔄 Power Query
* 📈 Data Visualization
* 📋 KPI Reporting
* 📌 Business Intelligence

---

## 📈 Key Insights

* Identified top-performing product categories.
* Highlighted high-revenue regions.
* Revealed profit trends across segments.
* Enabled dynamic filtering using slicers.
* Simplified executive reporting.
